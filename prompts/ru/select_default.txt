# Критерии отбора
1. Высокий score (приоритет 8+)
2. Разнообразие тем (не выбирай похожие моменты)
3. Нет пересечения по времени

# Правила
- Сохраняй start/end из кандидатов (можно немного подправить границы ±3 сек)

- Переоцени score при необходимости
